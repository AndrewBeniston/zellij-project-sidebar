<task type="auto">
  <name>Task 1: Create Rust WASM plugin project with ZellijPlugin implementation</name>
  <files>Cargo.toml, .cargo/config.toml, src/main.rs, zellij.kdl</files>
  <action>
Create four files from scratch to establish the complete plugin project:

1. **Cargo.toml** -- Package manifest:
   - name: "zellij-project-sidebar", version: "0.1.0", edition: "2021"
   - Dependencies: zellij-tile = "0.43.1", serde = { version = "1.0", features = ["derive"] }
   - Profile release: opt-level = "s", lto = true, strip = "debuginfo"

2. **.cargo/config.toml** -- Build target:
   - [build] target = "wasm32-wasip1"

3. **src/main.rs** -- Full ZellijPlugin implementation:
   - `use zellij_tile::prelude::*` and `use std::collections::BTreeMap`
   - `State` struct (derive Default) with `permissions_granted: bool` field
   - `register_plugin!(State)` macro call
   - `impl ZellijPlugin for State` with four methods:
     - `load()`: Call `request_permission()` with ReadApplicationState, ChangeApplicationState, Reconfigure. Call `subscribe()` with SessionUpdate and PermissionRequestResult. Log "Plugin loaded, requesting permissions" via `eprintln!`.
     - `update()`: Match on Event variants:
       - `Event::PermissionRequestResult(PermissionStatus::Granted)` -- set permissions_granted = true, eprintln "Permissions granted", return true
       - `Event::PermissionRequestResult(PermissionStatus::Denied)` -- eprintln "Permissions denied", return false
       - `Event::SessionUpdate(sessions, resurrectable)` -- eprintln session count (active + resurrectable), iterate sessions logging name, tab count, is_current_session. Return true.
       - `_ => false` (no re-render for unhandled events)
     - `render()`: If permissions_granted, println "Project Sidebar (loading...)". Else println "Waiting for permissions..."
     - `pipe()`: Return false (no-op stub for Phase 3)
   - CRITICAL: Use `eprintln!` for ALL logging. `println!` is ONLY for render output. Mixing these corrupts the plugin display.

4. **zellij.kdl** -- Development layout with hot-reload:
   - Tab bar plugin (borderless, size=1) at top
   - Vertical split: left 60% pane editing src/main.rs, right split horizontally with Cargo.toml editor above and plugin pane below (loading from file:target/wasm32-wasip1/debug/zellij-project-sidebar.wasm)
   - Status bar plugin (borderless, size=1) at bottom
   - Floating pane with develop-rust-plugin v0.3.0 from https://github.com/zellij-org/develop-rust-plugin/releases/download/v0.3.0/develop-rust-plugin.wasm
   - Set cwd to "." at layout root

Before creating files, ensure the wasm32-wasip1 target is installed: run `rustup target add wasm32-wasip1`.
