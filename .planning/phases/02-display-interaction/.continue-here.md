<decisions_made>

- Session-based over pin-based: default view shows only projects with active sessions, no manual curation needed
- Browse mode replaces old "show everything" approach: `/` opens it, Esc closes, Enter activates
- Attention uses pipe name encoding (sidebar::attention::session-name) rather than payload, matching zellij-attention's pattern
- is_primary flag prevents secondary plugin instances from overwriting global keybinds via reconfigure()
- Reddit post tone: casual, first-person, feedback-welcome, no marketing speak
</decisions_made>
