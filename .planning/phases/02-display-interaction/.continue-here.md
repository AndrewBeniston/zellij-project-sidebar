<decisions_made>

- Used `run_command()` for all host I/O (find, cat, sh -c printf) since WASM sandbox blocks direct filesystem access
- Manual JSON parsing instead of serde_json to keep binary small (~100KB)
- Fuzzy search uses subsequence matching (all query chars must appear in order in name)
- All bare letter keys go to search; commands on non-letter keys: Tab=pin, Delete=kill, Alt+R=rescan
- Ctrl+key combos don't work for commands (Zellij intercepts Ctrl+P for pane mode, etc.)
- Cmd+T rebound from `NewTab` to pipe message → plugin creates tab with inline KDL layout
- Section headers ("Pinned" / "Projects") only shown in discovery mode
- selected_index tracks position in filtered list, resolved to actual project index via filtered_indices()
- Esc has dual behavior: clears search first, deactivates sidebar second
- Session layout only applied for NotStarted projects; Running/Exited just switch_session()
</decisions_made>
