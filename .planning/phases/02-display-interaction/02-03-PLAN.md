**Test 1 - DISP-01 (Project list from config):**
- Look at the left sidebar pane
- Verify it shows the project names from the KDL config (e.g., help-self, tungsten-flow, etc.)
- If no projects appear, check `zellij.kdl` has project_N entries in the plugin block

**Test 2 - DISP-02 (Session status):**
- Projects with running Zellij sessions should show `*` or `>` prefix
- Projects with no session should show a space prefix
- Create a new session for one project (Test 5), then kill it (Test 7), verify status changes

**Test 3 - DISP-03 (Current session highlight):**
- The project matching the current Zellij session should show `>` prefix
- It should appear visually different (selected style from Zellij theme)

**Test 4 - INTR-01 (j/k navigation):**
- Press Alt+s to focus the sidebar
- Press j to move selection down, k to move selection up
- Verify the visual selection indicator moves
- Verify it stops at the top and bottom of the list (bounds clamping)

**Test 5 - INTR-02 + INTR-03 (Enter to switch/create):**
- Navigate to a project with no running session
- Press Enter -- verify a new Zellij session is created with the project's folder as cwd
- In the new session, run `pwd` to confirm the working directory matches the project path
- Switch back to the sidebar session, navigate to the session you just created
- Press Enter -- verify Zellij switches to that session

**Test 6 - INTR-04 (x to kill):**
- Navigate to a running session that is NOT the current session
- Press x -- verify the session is terminated and status updates to space or x
- Try pressing x on the current session -- verify it does NOT kill it (safety guard)

**Test 7 - INFR-04 (Focus management):**
- Press Esc to deactivate the sidebar
- Try pressing Tab to cycle through panes -- verify the sidebar is skipped
- Press Alt+s -- verify the sidebar becomes focused and selectable
- Press Esc again -- verify it becomes unselectable

What was built:
- Project list parsed from KDL config (project_0..N entries)
- Live session status indicators (> current, * running, x exited, space = none)
- Current session highlighting via Text.selected()
- j/k keyboard navigation with visual selection
- Enter to switch to running session or create new session with cwd
- x to kill non-current running sessions (with safety guard)
- Alt+s to activate sidebar (pipe-based focus)
- Esc to deactivate sidebar
- Plugin unselectable by default after permissions granted
